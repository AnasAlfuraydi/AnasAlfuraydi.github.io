\# Anas Alfuraydi — Data Analytics Portfolio



This repository contains my personal data analytics portfolio website, built to showcase my projects, technical skills, and business intelligence work.



\## Portfolio Overview



The website highlights selected projects in data analytics, business intelligence, dashboard development, and data-driven decision-making.



It includes Power BI dashboards, SQL/database projects, and future analytics projects using Python and other data tools.



\## Featured Projects

An interactive Power BI dashboard analyzing sales performance across executive, geographic, product, and customer perspectives.

A Power BI dashboard focused on sales performance, KPI cards, interactive slicers, and business-focused visual analysis.

A graduation project involving database design and an interactive GIS-based mapping application for exploring locations in Madinah.
